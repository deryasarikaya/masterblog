# Masterblog 📝

A simple blog application built with Flask and Python.

## Features
- View all blog posts
- Add new blog posts
- Edit existing blog posts
- Delete blog posts
- Like blog posts 👍

## Technologies
- Python
- Flask
- Jinja2
- HTML/CSS
- JSON

## How to run
1. Install Flask: `pip install flask`
2. Run the app: `python app.py`
3. Open your browser: `http://localhost:5000`
